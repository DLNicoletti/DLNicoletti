### 2. Regime Encoding (Φ(X))

<p align="center">
  <img src="Logos/ASHI-CORE/39EF8D8C-05B1-4F43-A99D-EC2C0D244B3E.gif" width="700">
</p>

**Description**

Following threshold crossing, the system undergoes a structural reorganization.  
The signal is no longer uniformly distributed but encoded into discrete spatial regimes.

- **Regime:** high-connectivity (K ≥ Kc)  
- **Behavior:** variance propagation and clustering  
- **Output:** feature-space transformation Φ(X)  

---

🧮 Core Metric ↓

The system is governed by a statistical order parameter:

K = σ / μ


where:

- **σ** → signal dispersion (variability)  
- **μ** → central tendency  

---

Critical Threshold ↓

Kc ≈ 1.441

- **K < Kc** → stable regime  
- **K ≥ Kc** → structural transition  

---

Conceptual Flow ↓

X(t) → Variance Amplification → K → Kc → Regime Transition → Φ(X)


---

Physical Interpretation ↓

ASHI-CORE does not model anomalies as isolated events.  
Instead, it captures **state transitions** driven by the redistribution of variability across the system.

- Stability → localized variance  
- Instability → global variance coupling  
- Transition → emergence beyond a critical structural limit.


ASHI-CORE formalizes the passage:

> from continuous, bounded dynamics  
> to discrete, structured regime encoding.
